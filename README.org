* Dependency
  1. WM: Sway
  2. Menu: Wofi
  3. Terminal: Alacritty
  4. File Manager: Thunar/nnn
  5. Wifi: NetworkManager
  6. Audio: Pulseaudio
  7. Font: JetBrainsMono Nerd Font
  8. Lock: Swaylock
  9. Screenshots : Grim/Slurp

   | Keybinding          | Function                |
   |---------------------+-------------------------|
   | Super+Enter         | Alacritty               |
   | Super+Shift+d       | Wofi                    |

   | Super+b             | Firefox                 |
   | Super+Shif+s        | Screenshots             |
   | Super+Shif+l        | Lock                    |
